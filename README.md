# freshservice-category-cleanup

## Setup

1. `git clone git@github.com:oliverba-unity/freshservice-category-cleanup.git`
2. `python3 -m venv .venv`
3. Activate the virtual environment
   1. Bash: `source ./venv/bin/activate`
   2. Fish: `source .venv/bin/activate.fish`
4. `pip install --requirement requirements.txt`
5. `cp .env.example .env`
6. `nano .env`
7. `python3 main.py`
